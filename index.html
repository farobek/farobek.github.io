About me

I am a Masters student in Computer Science. Topics that I am currently learning about include software engineering and web development with .NET technologies. Next semester, I will deepen my understanding of data structures and algorithms as well as learn about relational and object-oriented databases and human-computer interaction.

I develop software for the web, for smartphones and for the open-source community. I currently use AngularJS to build web applications, Java for mobile applications and Processing projects and Python for any software project that runs on a desktop environment. I have previously dabbled in Ruby to make a Rails web application. Many of my web and mobile applications use APIs to fetch data from the internet and all my projects are stored in my GitHub account.

My use of Java has enabled to develop a strong understanding Object Oriented Programming and building web applications with AngularJS has introduced to the MVC pattern. In order to diversify my skillset, I plan to dive in Haskell in the following months to develop an appreciation for the Functional Programming paradigm.
