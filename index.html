About me

I am a Computer Science student developing an Android app for a local charity on a volunteering basis. Modules that I have learned include software engineering and web development. This semester, I am deepening my understanding of data structures and algorithms as well as learning about machine learning, Big Data and intuitive human-computer interface design.

I develop software for the web, for smartphones and for the open-source community. I currently use AngularJS to build web applications and Java in Android Studio for mobile applications. I have previously dabbled in Ruby to make a Rails web application. Many of my web and mobile applications use web service APIs to fetch data from the internet and all my projects are stored in my GitHub account.

My use of Java has enabled me to develop a strong understanding of Object Oriented Programming and building web applications with AngularJS has given me exposure to the MVC pattern. I am currently learning NodeJS and I love it!
